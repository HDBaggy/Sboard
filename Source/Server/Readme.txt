1. Install the libwebsocket library in your machine
	- Linux : sudo apt-get install sudo apt-get install libwebsockets-dev

2. Download Socket server related code
	- https://github.com/mnisjk/cppWebSockets 
	- It has a bug that crashes the project so update the code as mentioned in - https://github.com/mnisjk/cppWebSockets/issues/3
	- Run any program from folder "example". Compile code using "make" command.

3. Download Socket client related code
	- https://github.com/dhbaird/easywsclient
	- Compile the sample program using "make" command.

4. The "Final" folder, install.sh file and make commands are very important.
